Developer Intern @ HCSC

Overview

In the summer of 2025, I had the priviledge of contributing as a Developer Intern within Product Engineering on the Enterprise API Team (EAT).

Project

EAT builds, manages, and facilitates the use of over 900+ applications across HCSC. Because of which, they have multiple projects for different product teams going on simeltaneously. Another intern and myself were tasked with finding an efficent way to displaying all this data for Product Teams and other stakeholders to see where this project is in our system.

We came up with a comprehensive card view, in which users can see all the projects effeciently. Each card represents a one project and displays miscellaneous information for that project, including: Title, Engagement Number, Project Code, Status, and Progress Bar (which represents the development progress). When the user clicks into the card, they can see a more detailed view with more information on that project. We were able to optimize this with advanced search and filter functionalities. Additionally, we were able to get our application to run in linear run time (originally it was running in O(n * (m + k) time)).
